# Simulador de Inversión - Método Monte Carlo
Este es un proyecto universitario basado en el libro de **Raúl Coss Bu**. Permite evaluar la viabilidad de un proyecto de inversión mediante 1,000 simulaciones de la TIR.

##  Características
- Interfaz gráfica moderna con **PyQt6**.
- Simulación de Monte Carlo para análisis de riesgo.
- Visualización de resultados mediante histogramas dinámicos (**Matplotlib**).
- Validaciones de seguridad para evitar errores de entrada.
